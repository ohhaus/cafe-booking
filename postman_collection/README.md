## Postman-коллекция для проверки API

Файл `Cafe_booking.postman_collection.json` содержит postman-коллекцию - набор подготовленных запросов для проверки работы API.

## Загрузка коллекции в Postman:

1. Запустите Postman.
2. В левом верхнем углу нажмите `File` -> `Import`.
3. Во всплывающем окне будет предложено перетащить в него файл с коллекцией либо выбрать файл через окно файлового менеджера.
Загрузите файл `Cafe_booking.postman_collection.json` в Postman.

## Подготовка к запуску коллекции:

1. В свойствах коллекции перейдите к Variables и заполните переменные:
 - normalUserTG  = 12345
 - adminUserTG = 54321
 - managerUserTG = 15243

Остальные переменные можно не заполнять т.к. они заполняются автоматически.
