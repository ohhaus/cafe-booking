# Проект бронирования столов в кафе

## Основное
1. Базовая версия Python - 3.12.x
2. Пакетный менеджер - **pip**
3. В файле `requirements_style.txt` находятся зависимости для стилистики
4. В каталоге `src` находится базовая структура проекта
5. В файле `src/requirements.txt` прописываются базовые зависимости


## Запуск проекта

### Через Docker Compose (рекомендуется)
Проект запускается через Docker Compose:
```shell
docker compose up
```

Для пересборки образов:
```shell
docker compose up --build
```

Для остановки:
```shell
docker compose down
```

### Локальный запуск FastAPI (если нужно)
Если требуется запустить только FastAPI приложение без Docker:
```shell
uvicorn src.main:app --reload
```

## Стилистика
Для стилизации кода используются пакеты `Ruff` и `Pre-commit`

### ⚠️ Обязательная установка Pre-commit
**Pre-commit хуки обязательны к установке!** После клонирования репозитория выполните:
```shell
pip install -r requirements_style.txt
pre-commit install
```

### Проверка стилистики
Проверка стилистики кода осуществляется командой:
```shell
ruff check
```

Если одновременно надо пофиксить то, что можно пофиксить автоматически, добавляем параметр `--fix`:
```shell
ruff check --fix
```

Автоматическая проверка и исправление стилистики при коммитах осуществляется через установленный pre-commit hook.
